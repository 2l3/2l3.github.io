What is a Twitter ID?

The Twitter ID is a unique value that every account on Twitter has. No two people have the same ID. Although an account can change its @handle, it can never change its Twitter ID.

How can you use TweeterID?

TweeterID allows you to easily look up any username (@handle) on Twitter and find out what their corresponding ID is. Alternatively, you can also use it to convert an ID into a username.

Twitter API with ID?

Using the TweeterID converter enables developers working with the Twitter API to easily transition between the primary key usage of the Twitter ID, and the publicly visible username (or @handle) of an account.
